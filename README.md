# SO-FIB

Coursework repository for **SO (Sistemas Operativos)** at the **Facultat d'Informàtica de Barcelona (FIB - UPC)**.

This repository contains operating systems exercises, example problems, and exam-oriented practice material.

## Overview

SO is one of the core systems subjects in the degree and provides the foundations for understanding processes, memory, concurrency, and the behavior of operating systems.

This repo is mainly focused on collecting useful practice material and solved exercises in an organized way.

## Contents

- Example exercises
- Exam preparation material
- Supporting code

## Focus

- Processes and threads
- Memory and resource management
- Synchronization
- Operating-systems fundamentals

## Notes

This repository is intended as a study and reference resource for the SO coursework at FIB.

Its value comes from clarity and usefulness as course material rather than from being a standalone project.
